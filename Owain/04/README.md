## Logical Operators

|                    |     Operators  |   Example |

| And            | &&                 |     a && b |
| Or               | ||                   |  a || b     |
| Not             |  !                    |  !a            |

Used a lot in conditional statements. e.g. you want to make sure a condition AND another condition are true.
